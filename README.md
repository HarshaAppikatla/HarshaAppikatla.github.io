## üõ†Ô∏è Customization Guide

### 1. Update Resume using the PDF file
1. Place your resume PDF file in this directory.
2. Rename it to `resume.pdf` (or update the filename in `index.html`).
3. In `index.html`, find the "Resume" button line:
   ```html
   <a href="#" class="btn btn-outline">... Resume</a>
   ```
   Change `href="#"` to `href="resume.pdf"`.

### 2. Update Social Links
Find the **Contact Section** in `index.html` and update the `href` attributes for LinkedIn, Twitter, etc.

### 3. Add More Projects
Copy the `project-card` article block in `index.html` and paste it to add more projects.
